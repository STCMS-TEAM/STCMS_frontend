How to use the app

Follow these steps to create or join tournaments and view results.

  1. Log in or sign up to save your tournaments and teams.
  2. Create a tournament from “Create Tournament” and set rules and participants.
  3. Add teams/players in “Manage Players & Teams” and set up matches.
  4. Track scores in “Track Live Results” and check the standings.
  5. Share the tournament link so others can join and see results.
